# Tony Hoang Projects

## Hazard and Collision Detection

Category: Computer Vision

Tony built a real-time ADAS hazard and collision detection system for NSW driving scenarios. The project detects brake lights, turn indicators, lanes, and potential hazards with stabilised inference.

Technologies: YOLOv8m, DeepSORT, CNNs, UNet, OpenCV, PyTorch, CUDA, CVAT, and Roboflow.

## IMDB Sentiment Analysis

Category: Natural Language Processing

Tony compared traditional machine learning and deep learning models on more than 50,000 IMDB reviews. Evaluation used ROC curves and confusion matrices.

Technologies: Python, NLTK, TF-IDF, Bag-of-Words, Logistic Regression, Naive Bayes, and Neural Networks.

## Human Activity Recognition

Category: Computer Vision

Tony implemented models for detecting and classifying human actions from video data, targeting applications in surveillance, healthcare, and smart environments.

Technologies: TensorFlow, PyTorch, video preprocessing, feature extraction, and deep learning models.

## NBA Rookie Career Longevity

Category: Data Mining

Tony predicted whether NBA rookies would remain in the league for five years using player attributes and performance data.

Technologies: Python, data mining, feature analysis, classification models, and performance evaluation.

## Virtual Library Platform

Category: Social Impact

Tony led a programming team to build a digital library platform providing textbook access for children in disadvantaged areas.

Technologies: Firebase, Java backend, HTML, CSS, API integration, and project coordination.
