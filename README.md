# Portfolio Full Stack App

React + TypeScript + Tailwind CSS frontend with a FastAPI backend.

## Frontend

```powershell
cd frontend
npm install
npm run dev
```

Replace `frontend/public/portrait.png` with your own portrait photo.
Add a faded Vietnamese place background as `frontend/public/vietnam-background.jpg`.

## Backend

```powershell
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
